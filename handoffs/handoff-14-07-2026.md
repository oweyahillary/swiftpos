## 1. ⭐ Priority (security — carried over from 13 Jul, NOT finished)

The repo was pushed public with a hardcoded Ed25519 tech-signing key (old commit
`cfd14a1`) and, per `PATCH_NOTES.md` (H2), some previously-hardcoded fallback secrets
in history. `techToken.ts` is now patched (no committed key; fail-fast in prod), and
local history was flattened to a single clean `Initial commit` — but the remote is
**not yet clean** and secrets are **not yet confirmed rotated**.

### 1a. Rotate exposed secrets — do this first, it's the only thing that neutralises the leak
Scrubbing git history does **not** help if someone already grabbed the key while it was
public. Rotate regardless of push state:
- **Supabase service-role key** — regenerate in Supabase dashboard (Project Settings → API).
- Fresh values in Render for: `JWT_SECRET`, `ADMIN_JWT_SECRET`, `SUPABASE_JWT_SECRET`,
  `TECH_HMAC_SECRET`, and SMTP / Resend password.
- **Tech signing key** — use the freshly generated Ed25519 pair only; never the old
  committed one. Set `TECH_SIGNING_PRIVATE_KEY` + `TECH_SIGNING_PUBLIC_KEY` in Render
  (full PEM blocks incl. BEGIN/END lines).

### 1b. Finish the clean push
Local `main` is one clean commit; remote still has old history. Last attempt hit a
`non-fast-forward` reject. Pick one:
- **Delete + recreate the repo** (thorough — actually wipes leaked commits off GitHub's
  servers): github.com as `oweyahillary` → repo Settings → Delete → recreate empty
  `swiftpos` → `git push -u origin main`.
- **Force-push** (faster, leaves old commit SHA-reachable a while):
  `git push -u origin main --force` then `git push origin --delete security-hardening`.
- Note: earlier 403 was a stale credential (`binaryonesolutions-blip`) not having write
  access; that appears resolved now (push reached the remote). If it recurs, clear the
  cached `git:https://github.com` entry in Windows Credential Manager.
- Consider keeping the repo **private** and enabling GitHub secret-scanning / push
  protection (Settings → Code security) so this is caught next time.
